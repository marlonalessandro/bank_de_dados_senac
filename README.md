## Descrição
Este projeto foi desenvolvido como atividade da disciplina de Banco de Dados, do curso de Análise e Desenvolvimento de Sistemas.  
O objetivo foi criar um banco de dados fictício para uma pizzaria, chamado **Pizzaria Bella**, que armazena informações de clientes, pizzas e pedidos.  
O projeto permite praticar comandos SQL de definição (DDL) e manipulação (DML) de dados.

## Objetivo
Aplicar comandos SQL de **DDL** (Data Definition Language) e **DML** (Data Manipulation Language), criando tabelas, relacionamentos e exemplos de manipulação de dados.  

## Estrutura do Banco de Dados
- **clientes**: armazena dados dos clientes (nome, telefone, cidade)  
- **pizzas**: armazena informações das pizzas (nome, categoria, preço)  
- **pedidos**: registra pedidos realizados, relacionando clientes e pizzas

## Como Executar

Se você quiser recriar ou testar o banco de dados:

1. Abra o MySQL Workbench (ou outro ambiente SQL que você use).  
2. Abra o arquivo do script SQL localizado em:
[sql_scripts/pizzaria_bella.sql](sql_scripts/pizzaria_bella.sql)
4. Execute o script para criar o banco de dados **Pizzaria Bella** com todas as tabelas e relacionamentos.

> Observação: lembrando que o banco de dados já está estruturado e funcional. Este passo é apenas para recriação ou para teste.
